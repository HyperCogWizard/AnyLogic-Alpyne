create_reinforcement_learning(inputs: Optional[alpyne.data.spaces.Configuration]) → alpyne.client.model_run.ModelRun[source]

Builds a new, single RL experiment using the provided configuration, or the default if none is provided.

Note: “the default” is all zeros/nulls (not your parameter’s defaults), so this should only be omitted if your logic handles this or if you have no Configuration.

Parameters

inputs – the initial configuration to start the model with

Returns

an object referencing the created model run
